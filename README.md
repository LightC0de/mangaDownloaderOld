# mangaDownloader
## О скрипте:
Скрипт "mangaDownloader" скачивает мангу с сайта http://readmanga.me и сохраняет её по папкам  
(шаблон: название_манги/volx/x/).  
  
<i>Над скриптом работают:</i>

</ul>
